# AWS Portfolio Site
Python portfolio site deployed on AWS EC2 Ubuntu server

# Live Demo
http://3.134.88.210:5000/

# Overview
This project is a personal portfolio website created with Python and Flask, then deployed on an Ubuntu server using Amazon EC2.

The goal of this project was to gain hands-on experience with:

-Cloud infrastructure using AWS
-Linux server management
-SSH remote access
-Python web development with Flask
-Web deployment and networking

# What I learned
Through this project I learned how to:

-Launch and configure an EC2 instance
-Use SSH keys to securely connect to a cloud server
-Manage Linux packages and Python environments
-Deploy a Flask web application
-Configure AWS security groups and open ports
